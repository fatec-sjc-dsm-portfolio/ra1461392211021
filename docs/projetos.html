API-1º Semestre

Durante o primeiro semestre de 2022, desenvolvemos uma página web que funciona como uma plataforma de terceirização de serviços para manutenção e resolução de problemas de hardware e/ou software. No back-end, utilizamos as tecnologias Python, Flask e MySQL para gerenciar funções e informações, enquanto o design e a estilização do front-end foram realizados com HTML, CSS e JavaScript. Nesse projeto, atuei como desenvolvedor back-end em um website para o controle de tickets de assistência técnica na FATEC, seguindo a metodologia Scrum. Minhas responsabilidades incluíram o desenvolvimento de CRUDs e a integração do front-end com o back-end para a exibição dos dados, utilizando Flask, Python, HTML, CSS e MySQL.
