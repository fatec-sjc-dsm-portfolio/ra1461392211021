1º Semestre - 404-BIS

Descrição do Projeto
Durante o primeiro semestre de 2022, desenvolvemos uma página web que funciona como uma plataforma de terceirização de serviços para manutenção e resolução de problemas de hardware e/ou software. No back-end, utilizamos as tecnologias Python, Flask e MySQL para gerenciar funções e informações, enquanto o design e a estilização do front-end foram realizados com HTML, CSS e JavaScript. Nesse projeto, atuei como desenvolvedor back-end em um website para o controle de tickets de assistência técnica na FATEC, seguindo a metodologia Scrum. Minhas responsabilidades incluíram o desenvolvimento de CRUDs e a integração do front-end com o back-end para a exibição dos dados, utilizando Flask, Python, HTML, CSS e MySQL.

Contribuições Pessoais

  • Atuei como desenvolvedor back-end em um sistema de controle de tickets de assistência técnica.
  • Desenvolvimento de funções CRUD para manipulação de dados.
  • Integração entre front-end e back-end utilizando Flask.
  • Utilização da metodologia Scrum para organização das entregas.

2º Semestre – Pay.me

Descrição do Projeto
O Pay.me é um projeto que buscou resolver os desafios enfrentados pelo cliente na gestão de processos financeiros e comerciais. Isso inclui dificuldades como cadastro manual de títulos e dados de endereço, geração manual de relatórios de clientes adimplentes e inadimplentes, complexidade no processamento de parcelas, formatação de dados de moeda, gestão de diferentes tipos de usuários e baixa manual de parcelas.

Contribuições Pessoais

  • Scrum Master na organização de sprints.
  • Desenvolvimento de rotas REST e testes automatizados.
  • Desenvolvimento das telas de login, cadastro e listagem.
  • Configuração de contêineres com Docker Compose.
