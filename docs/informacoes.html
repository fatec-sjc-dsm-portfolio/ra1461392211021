Minha História

O mundo dos jogos foi minha porta de entrada para a área de programação. Participei do desenvolvimento de um jogo de plataforma utilizando a ferramenta Construct 2, o que me proporcionou uma ótima experiência. Motivado por esse contato inicial, decidi cursar o ensino médio integrado ao técnico em 2019 na Etec Profª Ilza Nascimento Pintus, optando pelo curso de Desenvolvimento de Sistemas. Durante o curso, adquiri conhecimentos em diversas áreas, como Design Digital, Python, HTML, CSS, SQL, C++, JavaScript e React Native. Para a conclusão do curso, desenvolvi em grupo um jogo 3D educativo de exploração e aprendizagem sobre a fauna e flora da Amazônia, explorando tecnologias como C#, Unity, Blender e Mixamo.

Planos Futuros

Atualmente, estou continuando meus estudos na Fatec São José dos Campos - Prof. Jessen Vidal, no curso de Desenvolvimento de Software Multiplataforma. Participo de trabalhos utilizando a metodologia Scrum no API (Aprendizagem por Projeto Integrador), o que me permite expandir minha base de conhecimento. Além disso, estudo novas tecnologias por curiosidade, como automações e reconhecimento de imagens com Python.
